First launch

Opening TurboFiles the first time

TurboFiles is open-source and not yet code-signed, so your system asks you to confirm once. Here's the 10-second approval for each platform — after that, it opens like any other app.

macOS

  1. Double-click TurboFiles — a warning appears. Click Done.
  2. Open → System Settings → Privacy & Security.
  3. Scroll to Security, find "TurboFiles was blocked", click Open Anyway, then confirm with your password or Touch ID.

macOS 14 or earlier: Control-click the app → Open → Open.

Windows

  1. Run the installer — "Windows protected your PC" appears.
  2. Click More info.
  3. Click Run anyway, then install as usual.

SmartScreen stops warning once the app builds reputation.

Linux

  1. AppImage: right-click → Properties → allow executing (or chmod +x).
  2. Double-click to run — no system prompt.
  3. .deb / .rpm: install with your package manager as normal.

Most desktops run AppImages straight away.

Why the prompt? Signing certificates cost money to maintain. The entire app is public on GitHub, so you can verify exactly what you're running.
